def cs110graphics.RunWithYieldDelay (   window,
  func 
)

Begins an animation loop.

Parameters:
window - Window
func - function which returns a generator of int

The function given must use yield statements to indicate moments in the code when the system should stop and refresh the window. The system will pause for the number of milliseconds given to yield. This allows for the creation of animation systems by refreshing the window between movements.
